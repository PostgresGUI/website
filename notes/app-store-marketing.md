# PostgresGUI - App Store Marketing Content

## Subtitle

Lightweight and navtive PostgreSQL client for Mac

## Promotional Text

A lightweight PostgreSQL client for macOS. Manage connections, browse databases, execute SQL queries with syntax highlighting, and view results—all in a native SwiftUI interface.

## Description

PostgresGUI is a lightweight PostgreSQL client built specifically for macOS. Designed for users learning PostgreSQL and developers who need quick database access, it provides essential database management capabilities in a clean, native macOS interface.

**Connection Management**

- Save and manage multiple PostgreSQL connection profiles
- Secure password storage using macOS Keychain
- Quick-connect to localhost or remote servers
- Test connections before saving
- Switch between saved connections instantly

**Database & Table Browsing**

- Visual navigation of all databases and tables
- Create and delete databases directly from the interface
- Browse table schemas and structures
- View data from any table with a single click
- Filter out system tables for cleaner navigation

**SQL Query Editor**

- Full-featured SQL editor with syntax highlighting
- Execute queries with keyboard shortcuts (Cmd+Enter)
- View query results in a sortable, interactive table
- Support for all PostgreSQL data types including timestamps, booleans, and NULL values
- Real-time execution metrics showing row counts and query time

**Data Viewing & Analysis**

- Paginated results for large datasets
- Sortable columns in ascending or descending order
- Automatic data type detection and formatting
- Copy data values with text selection
- Clear display of empty results and column headers

**Native macOS Experience**

- Built with SwiftUI for modern macOS design
- Liquid Glass UI elements for a beautiful interface
- Three-column split view optimized for database workflows
- Resizable panes for customizing your workspace
- Full light and dark mode support
- Keyboard shortcuts for common operations

PostgresGUI is perfect for developers learning PostgreSQL, students working on database projects, or anyone who needs a straightforward tool for managing Postgres databases on Mac. No complicated setup, no heavy resource usage—just a clean, efficient PostgreSQL client that feels right at home on macOS.
